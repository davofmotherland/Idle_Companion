# Product Inputs

## Current Decisions

| Item | Decision |
| --- | --- |
| Temporary project name | Project Seal |
| Default pet direction | White baby seal |
| First release language | Bilingual |
| Expected Steam price | About USD 1 |

## Product Position

Project Seal should feel like a low-friction, cute, offline desktop companion. The default character should be visually readable as a white baby seal even at small desktop sizes.

## Immediate Implications

- Art references should focus on round white baby seals, soft faces, short flippers, simple dark eyes, and readable silhouettes.
- The initial palette should stay light but not pure white, so the character remains visible on bright desktops.
- Bilingual support should be designed from the first UI pass rather than added after text is scattered through the app.
- The USD 1 target means the MVP must stay lean: strong default character, stable desktop behavior, and a clear customization hook matter more than content volume.
