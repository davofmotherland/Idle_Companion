# What The User Must Prepare

## Required Before Development Can Run

1. Install Node.js LTS from the official Node.js website.
2. Install Git for Windows.
3. Restart the Codex app or terminal after installation so `node`, `npm`, and `git` are available.

Verify in the project folder:

```powershell
node --version
npm --version
git --version
```

## Required Product Inputs

1. Temporary project name.
2. Main default pet direction: small person, cat, dog, robot, ghost, or slime.
3. First release language: Chinese, English, or bilingual.
4. Desired Steam price range.

## Reference Assets To Add

Put files in these folders:

- Default character references: `assets/references/default-character/`
- Photo-to-pixel test images: `assets/references/photo-to-pixel-tests/`
- Bed, toy, and food references: `assets/references/scene-props/`
