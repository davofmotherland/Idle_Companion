# Asset Intake Checklist

## Needed From User

Default pet:

- Main direction is set: white baby seal.
- Add 1-3 white baby seal style references under `assets/references/default-character/`.

Photo-to-pixel tests:

- Add 3-5 test images under `assets/references/photo-to-pixel-tests/`.
- Include at least one face/person, one pet, and one object if possible.

Scene props:

- Add references for bed, toy, and food/snack under `assets/references/scene-props/`.

Audio:

- Add temporary sound references or licensed effects under `assets/sounds/`.

Models:

- Do not add large local AI models yet unless requested. Keep candidates or notes under `assets/models/`.

Project metadata:

- Temporary project name: Project Seal.
- Target language for first release: bilingual.
- Desired Steam price range: about USD 1.

## License Rule

References are acceptable for direction, but final shipped assets must be original, licensed, or generated from models/assets with commercial permission.
