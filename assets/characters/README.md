# Character Assets

Put character packs here.

Recommended layout:

```text
characters/
  default_pet/
    manifest.json
    default_pet_sheet.png
    default_pet.aseprite
    preview.png
```

MVP animation target:

- idle
- blink
- walk
- run
- sit
- sleep
- happy
- angry
- dragged
- fall
- land
- poke
